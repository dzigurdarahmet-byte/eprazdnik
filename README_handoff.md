# eprazdnik-site

Кастомный сайт-каталог для агентства «Е-Праздник». MVP в разработке через Claude Code.

## Запуск Claude Code

В этой папке открой Claude Code и скорми задачу:

> Прочитай файл CC_megaprompt_MVP_v2.md в этой папке и выполни задачу согласно мегапромта.
> Прототип-референс лежит в proto/. Notion-токен и ID баз — в .env.local (НЕ читай и не логируй токен).
> Действуй по этапам §6-10 мегапромта, после каждого этапа делай коммит.

## Артефакты в папке

- `proto/` — HTML-прототип (статический mockup, референс для вёрстки и стилей)
- `CC_megaprompt_MVP_v2.md` — основное задание для CC
- `Кастомный_сайт_Scope_v1.md` — контекст по продукту
- `.env.local` — секреты (НЕ коммитить, лежит в .gitignore)
- `.env.example` — шаблон env для других разработчиков

## Дальнейшие шаги

После выполнения CC отчитается по формату §10 мегапромта:
- ссылка на GitHub репо
- Production URL на Vercel
- Lighthouse результаты
- Acceptance criteria статус
